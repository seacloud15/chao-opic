# ChaoOPIc

부서원들의 베트남어 OPIc 등급 취득을 효율적으로 지원하기 위한 시험 시뮬레이션 및 학습 보조 프로그램

## Tech Stack

- HTML5 / CSS3 / JavaScript

## Getting Started

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## Project Structure

```
chao-opic/
├── index.html        # 메인 페이지
├── css/style.css     # 스타일
├── js/main.js        # 자바스크립트
├── images/           # 이미지 파일
└── docs/             # PDCA 문서
```
