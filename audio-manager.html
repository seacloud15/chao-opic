🗂️ 학습 자료 관리

⚠️ 브라우저 미지원
이 기능은 Chrome, Edge 등 최신 브라우저에서만 지원됩니다.
File System Access API가 지원되는 브라우저를 사용하세요.
📂 오디오 파일 인덱스가 없습니다
시뮬레이션을 사용하려면 먼저 오디오 파일이 있는 audio/simulation 폴더를 선택해야 합니다.

📁 폴더 구조 예시:

audio/simulation/
├── self-introduction/
├── selfAssess/
├── survey/
│ ├── 영화보기_01/
│ ├── 공원가기_01/
│ └── ...
├── non-survey/
├── rolePlay/
└── issueComparison/
✅ 오디오 인덱스가 준비되었습니다
시뮬레이션에서 정상적으로 문제를 생성할 수 있습니다.

📊 인덱스 통계

폴더 스캔 중...
오디오 파일을 검색하고 인덱스를 생성하고 있습니다. 잠시만 기다려주세요.

💡 사용 안내

  1. 폴더 선택: "오디오 폴더 선택 및 스캔" 버튼을 클릭하여 프로젝트 안의 audio 폴더를 선택합니다.
  2. 자동 스캔: 선택한 폴더 내의 모든 오디오 파일을 자동으로 스캔하여 인덱스를 생성합니다.
  3. 캐시 저장: 생성된 인덱스는 브라우저의 localStorage에 저장되어 다음번에 다시 선택할 필요가 없습니다.
  4. 시뮬레이션 사용: 인덱스가 준비되면 "오픽 시뮬레이션" 페이지에서 정상적으로 문제를 생성할 수 있습니다.
  5. 파일 변경 시: 오디오 파일을 추가/삭제/변경한 경우 "다시 스캔" 버튼을 눌러 인덱스를 업데이트하세요.
